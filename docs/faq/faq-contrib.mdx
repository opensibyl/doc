# 我想开发该项目

感谢你对这个项目感兴趣！

## 环境

由于 tree-sitter 的存在，我们需要 cgo 来编译。参考官方教程在机器上安装 g++/gcc 即可。

## 编译

我们使用 make 来打出二进制包。

```
make
```

如果你不喜欢，亦可结合 makefile 中的逻辑，自行编写脚本。

## 模块与代码

目前我们重点开发的模块有两个：

- 更多的语言支持：[extractor](https://github.com/opensibyl/sibyl2/tree/master/pkg/extractor)
- 更多丰富的接口：[server](https://github.com/opensibyl/sibyl2/tree/master/pkg/server)

为节约您的时间，建议先通过 issue 面板沟通。
