# 支持的语言

| Languages  | Function | Function Context | Class |
|------------|----------|------------------|-------|
| Golang     | Yes      | Yes              | Yes   |
| Java       | Yes      | Yes              | Yes   |
| Python     | Yes      | Yes              | Yes   |
| Kotlin     | Yes      | Yes              | Yes   |
| JavaScript | Yes      | Yes              | Yes   |

基于 [tree-sitter](https://tree-sitter.github.io/tree-sitter/)，理论上我们能够支持几乎所有的编程语言。
如果您对新语言有需求，可参考 [如何开发](./faq-contrib) 一节。它并不复杂。

与传统工具不同，我们的服务端与采集端做了彻底的解耦。
在完成采集端支持之后，服务端不需要额外修改即可完全兼容。
