# 支持的语言

| Languages | Function | Function Context | Class   |
|-----------|----------|------------------|---------|
| Golang    | Yes      | Yes              | Yes     |
| Java      | Yes      | Yes              | Yes     |
| Python    | Yes      | Yes              | Yes     |

基于 [tree-sitter](https://tree-sitter.github.io/tree-sitter/)，理论上我们能够支持几乎所有的编程语言。如果您对新语言有需求，可参考 [如何开发](./faq/faq-contrib) 一节
