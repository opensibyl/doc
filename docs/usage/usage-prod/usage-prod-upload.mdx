# 上传

在部署完成后，您即可一键上传您的仓库。

在仓库目录下执行：

```
./sibyl upload
```

看到下列日志即可代表上传完成。

```

{"level":"info","ts":1670140952.0042171,"caller":"upload/cmd_upload.go:93","msg":"upload finished"}
```
