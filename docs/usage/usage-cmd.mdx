# 使用 - 命令行

命令行是最简单的使用该项目的方式。我们认为应该开箱即用的功能都会包含在此处。
如果你只是想在一些离线场景中使用，可以使用该模式。

## 下载

你可以在 release 页面上下载到你需要的二进制文件。

https://github.com/williamfzc/sibyl2/releases

## 使用

### 快照功能

```
./sibyl extract --src . --output hello.json
```

只需要几秒：
