直接使用命令行

命令行只会包含我们早期内置的功能，通常情况下他将直接产出 json 文件作为输出。您需要自己进行处理。

下载​

你可以在 release 页面上下载到你需要的二进制文件。

https://github.com/opensibyl/sibyl2/releases

使用​

快照功能​

./sibyl extract --src . --output hello.json

只需要几秒：

$ ./sibyl extract --src . --output hello.json
