# 直接使用 golang api

你可以像正常使用 go 包一样使用它，开展更为个性化的需求搭建。

```bash
go get -u github.com/opensibyl/sibyl2
```

## Work with 标记系统 （0.15.0之后）

在 0.15.0 之后，sibyl2将统一使用标记系统来满足各种业务域需求。详见 [标记系统](./tag-system)

## 如果想轻度定制

例如为您的团队封装一个公司内部使用的上传器，可以参考：https://github.com/opensibyl/git-connector 。

## 如果想深度定制

例如只想利用 sibyl2 的推理功能，可以参考源码 `cmd/sibyl/subs/diff` 目录下，我们的离线版diff。
